Say Hello, to the Quartz UI Kit

Simple, concise, flat, and open source.

Design

Flat Design

Using a flat design, Quartz communicates, simple, and concise information.

Ease of Use

Providing an easy way to build UIs with Quartz has been a strong point for the development of the kit.

Column Based

Quartz uses a column based system to make building responsive sites, even easier.
